Open MS.exe to play the demo.  

Press Alt+F4 to close window within the demo.  

Controls:  
    - Keyboard & Mouse:  
        - W = Throttle  
        - S = Brake  
        - A = Steer Left  
        - D = Steer Right  
        - Space = Handbrake  
        - Mouse = Move Camera  
        - Left Shift = Pitch Up (while airborne)  
        - Left Ctrl = Pitch Down (while airborne)  

    - Gamepad:
        - Right Trigger = Throttle  
        - Left Trigger = Brake  
        - Left Stick = Steer (and pitch while airborne)  
        - Right Bumper = Handbrake  
        - Right Stick = Move Camera  

Note: if car is upside down you can steer left or right to flip it over.
